# fraud-detection_intern_pro

## 🎯 Credit Card Fraud Detection System

Enterprise-level ML web application for real-time credit card fraud detection with professional UI/UX and advanced analytics.

### 🚀 Features
- **Real-time fraud detection** using scikit-learn RandomForestClassifier
- **Professional UI** with gradient animations and glass-morphism effects
- **Interactive charts** with Chart.js visualization
- **Multiple input methods** (sample data, manual entry, custom values)
- **Risk analysis dashboard** with confidence scores

### 🛠️ Technologies
- **Backend:** Flask, scikit-learn, numpy
- **Frontend:** HTML5, CSS3, JavaScript, Chart.js
- **ML Model:** RandomForestClassifier with 12-feature input

### 📋 How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run Flask server: `python app.py`
3. Open browser: `http://localhost:5000`

### 🎨 Features
- **Generate Sample Data** - Auto-generates realistic transaction data
- **Manual Transaction Entry** - Input amount and time manually
- **Real-time Analysis** - Instant fraud detection with risk percentages
- **Interactive Dashboard** - Visual risk analysis with charts
